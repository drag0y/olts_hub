### Включение/Выключение CATV порта
##### GPON
> 1.3.6.1.4.1.2011.6.128.1.1.2.63.1.2

Делается через snmpset\
id порта + id ОНУ, далее номер порта CATV и i 1
Пример:
> snmpset -v2c -c private 192.168.1.2 1.3.6.1.4.1.2011.6.128.1.1.2.63.1.2.4227870464.0.1 i 1

Для выключения порта, используется тот же oid, но в конце двойка вместо единицы.

### Удалить ОНУ
Делается через snmpset
##### EPON
> 1.3.6.1.4.1.3320.101.11.1.1.2

Здесь к oid нужно прибавить id порта и мак ОНУ в десятичном формате.\
Пример:
> snmpset -v2c -c private 192.168.1.2 1.3.6.1.4.1.3320.101.11.1.1.2.15.28.135.121.23.207.189 i 0

##### GPON
